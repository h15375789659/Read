# 语言偏好设置

请始终使用**中文**回答用户的问题和进行交流。

## 规则

- 所有回复、解释、注释都使用中文
- 代码注释优先使用中文
- 技术术语可以保留英文原文，但解释用中文
- 错误信息和日志分析用中文说明

# 测试文件管理

## 规则

- 当属性测试（PBT）或单元测试执行完成并通过后，删除对应的测试文件
- 删除测试文件前确保测试已成功通过
- 保留测试结果记录，但移除测试源代码文件

# 任务执行偏好

## 规则

- 在进行新任务时不打开新的对话窗口，在当前窗口继续执行
- 保持对话的连续性，在同一上下文中完成相关任务
- 当一个问题进行第一次修复后未修复成功时，第二次应回退第一次的改动并添加日志函数，跟用户说明操作以及应当查询的日志，在成功修复后应删除添加的日志函数
- 进行新任务时，由于已经在原本的基础上改了很多代码，跟原计划不太一样了，所以应该把计划告诉用户，并且要查看相关的代码，进行可行性分析，要求为尽量在不影响原来的功能完成新任务，得到用户的同意之后才能进行任务的完成。
